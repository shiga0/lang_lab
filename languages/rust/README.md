# Rust リファレンス

## 概要

| 項目 | 内容 |
|------|------|
| 登場年 | 2015 (1.0) |
| 設計者 | Graydon Hoare (Mozilla) |
| 現在の開発 | Rust Foundation |
| 型システム | 静的・強い・型推論あり |
| メモリ管理 | 所有権システム (GCなし) |
| パラダイム | マルチ (手続き/OOP/FP) |
| 主な用途 | システム, CLI, WebAssembly, 組み込み |

## 特徴

### 1. 所有権システム
- GCなしでメモリ安全性を保証
- コンパイル時にデータ競合を防止
- ゼロコスト抽象化

### 2. 型システム
- 強力な型推論
- ジェネリクスと Trait
- 代数的データ型 (enum)

### 3. エラー処理
- `Result<T, E>` と `Option<T>`
- 例外なし（panic は回復不可能）
- `?` 演算子でエラー伝播

### 4. 並行処理
- 「恐れなき並行性」
- Send/Sync トレイトで安全性保証
- async/await サポート
